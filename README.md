# e-Therm Plus KS - Home Assistant Add-on Repository

Questo repository contiene add-on personalizzati per Home Assistant (Hass.io).

## Add-on disponibili
- `e_therm_plus_ks`: e-Therm Plus KS

## Installazione
1. In Home Assistant vai su **Impostazioni -> Componenti aggiuntivi -> Add-on Store**.
2. Apri il menu (in alto a destra) e scegli **Repositories**.
3. Aggiungi questo URL del repository.
4. Installa l'add-on `e_therm_plus_ks`.

## Note
- Le icone e i loghi sono dentro la cartella dell'add-on.
- Per dettagli di configurazione, vedi il README dell'add-on.
